## 빠른 시작

Claude Code 에서 아래 두 줄이면 설치 끝.

```bash
# 1. 마켓플레이스 등록 (최초 1회)
/plugin marketplace add bluei98/haru-skills

# 2. 원하는 플러그인 설치
/plugin install domain-search@haru-skills
```

> [!NOTE]
> Claude Code 플러그인 시스템은 `<plugin>@<marketplace>` 형식의 설치 문법을 요구합니다.
> `@haru-skills` 를 반드시 붙여주세요.
